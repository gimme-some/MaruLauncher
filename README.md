<h1 align="center">Maru Launcher</h1>

<p align="center">마루의 마인크래프트 서버 전용 런처예요.<br>Java도, Forge도, 모드도 신경 쓸 필요 없어요 — 런처가 알아서 다 해드립니다.</p>

---

## ✨ 이런 걸 해줘요

- 🧩 **모드·파일 자동 설치** — 서버에 필요한 모드와 파일을 알아서 받아 설정해요. 실행 전 파일을 검증하고, 손상되거나 빠진 파일은 다시 받아요.
- ☕ **Java 자동 설치** — Java가 없어도, 버전이 안 맞아도 런처가 알맞은 걸 깔아줘요.
- 🔒 **Microsoft 계정 로그인** — 여러 계정을 추가해 쉽게 전환할 수 있어요. 비밀번호는 저장되지 않아요.
- 🔄 **자동 업데이트** — 새 버전이 나오면 런처가 스스로 업데이트해요.
- 🎮 **PLAY 한 번**이면 접속 끝. 선택한 서버의 접속자 수도 보여줘요.

> 🎯 Minecraft **1.20.1**

---

## ⬇️ 다운로드

최신 버전은 [**Releases**](https://github.com/gimme-some/MaruLauncher/releases/latest) 에서 받으세요.

자기 환경에 맞는 파일 **하나만** 받으면 돼요:

| 환경 | 받을 파일 |
| --- | --- |
| 🪟 **Windows** | `Maru-Launcher-setup-VERSION.exe` |
| 🍎 **Mac (Apple Silicon · M1~M5)** | `Maru-Launcher-setup-VERSION-arm64.dmg` |
| 🍎 **Mac (Intel)** | `Maru-Launcher-setup-VERSION-x64.dmg` |

> 🔎 **내 맥 칩 확인:**  메뉴 → *이 Mac에 관하여* → 칩이 *Apple M~* 면 arm64, *Intel* 이면 x64

### 처음 실행 시 경고가 떠요 (정상이에요)

개인이 만든 런처라 코드 서명이 없어서, 처음 실행할 때 경고가 떠요:

- **Windows:** "Windows의 PC 보호" 창 → **추가 정보** → **실행**
- **Mac:** 실행이 막히면 → **시스템 설정 → 개인정보 보호 및 보안 → "그래도 열기"**

---

## 🚀 시작하기

1. 런처를 설치하고 실행하기
2. **Microsoft 계정**으로 로그인
3. **PLAY** → 모드·파일 자동 설치 → 서버 입장! 🌞
